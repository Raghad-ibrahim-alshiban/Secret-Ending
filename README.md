# The Wedding Surprise Quest 💍✨

## Overview
This is a 3D Single-Level Adventure Game created as a special surprise for a friend's wedding announcement. The project was built in Unity with a unique heart-warming backstory: A friend reached out to me with a surprise mission to create a game where her sister could announce her marriage to her best friends in a fun and interactive way.

## The Challenge 🚀
I was challenged to build this game from scratch in just 10 hours! As a beginner, this was a massive undertaking. I managed to design the 3D environment, script the mechanics, and handle the UI within that impossible timeframe. This project is a testament to my dedication to learning and delivering under extreme pressure.

## Gameplay & Features
The game consists of one intense 3D level where players must:
- Beat the Clock: A countdown timer adds pressure to the mission.
- Search & Explore: Inspect various boxes and containers in a 3D space to find the hidden key.
- Interaction: Use the "E" key to interact with objects and open the final door.
- The Big Reveal: Upon finding the key and reaching the end, a celebratory "Wedding Reveal" screen pops up with a special musical zaffah.

## Technical Details
- Engine: Unity 3D
- Perspective: First-Person 3D
- Scripting: C#
- Structure: Single Level (One-room/Area challenge)
